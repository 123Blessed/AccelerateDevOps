### Sourcecode

_Javascript:_

```js
function writeHelloWorld(name) {
  console.log(`Hello World and ${name}!`);
}
writeHelloWorld('Jon Doe');
//=> prints 'Hello World and Jon Doe!' to STDOUT.
```

_Ruby:_

```ruby
def print_helloWorld(name)
  puts "Hello World and #{name}!"
end
print_helloWorld('Jon Doe')
#=> prints 'Hello World and Jon Doe!' to STDOUT.
```
